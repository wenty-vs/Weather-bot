# Weather-bot
## 🚀 Основні функції
* **Миттєвий пошук:** Отримання температури (поточної, відчутної, мін/макс) та опису стану неба.
* **Інтеграція з OpenWeatherMap API:** Робота з зовнішнім сервісом через REST-запити.
* **Система логування:** Бот автоматично сповіщає адміністратора про нові запити користувачів, що дозволяє моніторити активність.
* **Локалізація:** Повна підтримка української мови у відповідях.
